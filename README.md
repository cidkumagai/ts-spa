

# TypeScript Reactを用いたSPAです。
 
1.RESAS-APIとの通信を行い、都道府県一覧を取得  
2.取得した都道府県一覧を基にチェックボックスを生成  
3.選択された都道府県の人口推移情報を取得し折線グラフを生成  

 こちらは[github pages](https://cidkumagai.github.io/ts-spa/)で公開しています。
